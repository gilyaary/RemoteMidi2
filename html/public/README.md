To start the Java code:
./gradlew bootrun

Then the API could be access with Swagger:
http://localhost:8080/swagger-ui.html#/hello-controller/indexUsingGET

 

 The VUE UI is in:
 <root>/html/public/vue-files/vue-project-1

 And can be served with:
 yarn serve


 And then viewed:
 http://localhost:8081/
